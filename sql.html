Advanced SQL Concepts Demonstrated

  • Common Table Expressions (CTEs): Simplified complex queries and allowed modular, readable code for multi-step transformations.
  • Window Functions: Used ROW_NUMBER(), RANK(), LEAD()/LAG(), SUM() OVER() to calculate running totals, rankings and comparisons without extra joins.
  • Complex Joins: Leveraged INNER, LEFT, RIGHT, FULL OUTER and CROSS joins for multi-table analysis across normalised datasets.
  • Pivoting and Unpivoting: Transformed rows into columns and vice versa to produce aggregated views suitable for reporting dashboards.
  • Nested Subqueries & Conditional Aggregations: Applied CASE statements and filtered aggregations for targeted metrics and KPIs.
  • Query Optimisation: Indexed key columns, analysed execution plans and optimised query performance on millions of rows.
